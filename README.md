###  简易安装
**wget -qO- https://raw.github.com/VZIKL/vimrc/master/install.sh | sh -x**


####   快捷键
1. C-i,Tab 编译运行文件
2. kj=ESC
3. 命令模式下H,L移动到行首和行尾
4. 底线命令模式中 C-a到行首 C-e到行尾
5. C-x 改变左侧行号
6. F3=NerdTree F8=Tagbar
7. F6关闭,打开文件高亮
8. C-w 切换窗口
9. C-p 查找文件
