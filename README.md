# Vim Config

## Install

> wget https://raw.githubusercontent.com/VZIKL/vimrc/master/install.sh | sh -c

### Update Plugin

> cd ~/.vim && git submodule update --remote

### Key Bindings

---

| Keybinds | Description              |
| -------- | ------------------------ |
| kj       | ESC                      |
| F3       | Open NerdTree            |
| F5       | compile file             |
| C-h      | autoformat file          |
| H        | To beginning of the line |
| L        | To ending of the line    |
| U        | Redo                     |
| gb       | Next Buffer              |
| gB       | Last Buffer              |
